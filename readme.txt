Emoji Keyboard v1.0
Developed by (C) Rafsun Masud Prince

An on-screen virtual keyboard for emojis. Supports thousand popular emoji.

Tips:
1. Press F8 to toggle system tray mode.
2. Press filter button to filter emojis by popular sites and services.
3. Change window transparency in settings.
4. Right click on emoji to copy it to clipboard.
